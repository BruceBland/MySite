# Bruce Bland Website

Apple-inspired single-page profile site for **Bruce Bland**, built as a static project for GitHub Pages.

## Files

- `index.html` - page structure and profile content
- `styles.css` - visual style, responsive layout, Apple-inspired glass/gradient look
- `script.js` - reveal-on-scroll animation

## Run locally

Open `index.html` in your browser.

## Publish on GitHub Pages

1. Push this repository to GitHub.
2. In GitHub, go to **Settings -> Pages**.
3. Under **Build and deployment**, set:
   - **Source**: `Deploy from a branch`
   - **Branch**: `main` (or your default branch), folder `/ (root)`
4. Save and wait for deployment.
